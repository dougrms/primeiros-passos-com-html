O que é HTML?

HTML, que significa HyperText Markup Language, é a linguagem padrão usada para criar e estruturar páginas na web. Pense no HTML como a planta de uma casa. Ele define a estrutura básica, como onde ficam as paredes, portas e janelas, mas não decide a cor das paredes ou o tipo de móveis. HTML organiza o conteúdo de uma página, como textos, imagens e links, para que os navegadores possam exibi-los corretamente.

Estrutura Básica de um Documento HTML:

Todo documento HTML começa com uma estrutura básica que ajuda os navegadores a entenderem como mostrar o conteúdo da página. Aqui está um exemplo simples:

Explicação da estrutura:

  • < !DOCTYPE html >

    Esta declaração informa ao navegador que ele está lidando com um documento HTML5. É essencial para garantir que a página seja renderizada corretamente em todos os navegadores modernos.

  • < html lang="pt-BR" >

    A tag <html> é a raiz do documento HTML. O atributo lang define o idioma do conteúdo, que neste caso é português do Brasil. Isso ajuda os mecanismos de busca e leitores de tela a entenderem o idioma da página.

  • < head >

    Esta seção contém metadados sobre o documento, como o título da página, links para arquivos CSS, scripts e outras informações que não são exibidas diretamente na página.

  • < meta charset="UTF-8">

    Define a codificação de caracteres do documento como UTF-8, que suporta praticamente todos os caracteres de escrita do mundo. Isso garante que caracteres especiais sejam exibidos corretamente.

  • < title >

    Define o título da página que aparece na aba do navegador. É importante para SEO, pois os motores de busca usam o título para indexar a página.

  • < body >

    Contém todo o conteúdo visível da página, como textos, imagens e links. É onde você coloca tudo o que quer que o usuário veja.

O que é uma Tag HTML?

Uma tag HTML é um elemento fundamental da linguagem HTML, usado para criar e estruturar o conteúdo de uma página da web. As tags são como "rótulos" que dizem ao navegador como exibir o conteúdo. Elas são escritas entre colchetes angulares (<>) e geralmente vêm em pares: uma tag de abertura e uma tag de fechamento.

Estrutura de uma Tag HTML

  • Tag de abertura: Indica o início de um elemento.

    • Exemplo: <p>

  • Conteúdo: O texto ou elementos que a tag envolve.

    • Exemplo: Este é um parágrafo.

  • Tag de Fechamento: Indica o fim de um elemento.

    • Exemplo: </p>

  • Exemplo completo:

    • <p>Este é um parágrafo.</p>

Atributos

Tags HTML podem ter atributos, que fornecem informações adicionais sobre o elemento. Os atributos são escritos dentro da tag de abertura e geralmente vêm em pares de nome e valor.

  • Exemplo de uma tag com atributo:

    • <a href="https://www.exemplo.com" title="Visite nosso site">Clique aqui</a>

    • href: Define o destino do link.
    • title: Fornece uma dica de ferramenta que aparece quando o usuário passa o mouse sobre o link.

Principais Tags HTML e seus atributos:

    < link > :

    • O que faz: Conecta a página a recursos externos, como folhas de estilo CSS.
    • Por que é importante: Permite que você adicione estilos e funcionalidades à sua página.
    • Principais Atributos:
      • href: Especifica o URL do recurso vinculado.
      • rel: Define a relação entre o documento atual e o recurso vinculado. Exemplo: rel="stylesheet".

    <h1> a <h6> :

    • O que faz: Define cabeçalhos, do mais importante <h1> ao menos importante <h6>.
    • Por que é importante: Ajuda a estruturar o conteúdo e é importante para SEO.
    • Atributos: Podem usar atributos globais como class e id.

    <p> :

    • O que faz: Define um parágrafo de texto.
    • Por que é importante: Usado para blocos de texto corrido.
    • Atributos: Podem usar atributos globais como class e id.

    <a> :

    • O que faz: Cria um link para outra página ou recurso.
    • Por que é importante: Permite a navegação entre páginas.
    • Principais Atributos:
      • href: Define o destino do link.
      • title: Fornece uma dica de ferramenta que aparece quando o usuário passa o mouse sobre o link.
      • target: Define onde abrir o link. Exemplo: target="_blank" para abrir em uma nova aba.

    <img> :

    • O que faz: Insere uma imagem na página.
    • Por que é importante: Adiciona elementos visuais ao conteúdo.
    • Principais Atributos:
      • src: Especifica o caminho para a imagem.
      • alt: Fornece um texto alternativo para a imagem, importante para acessibilidade.
      • width e height: Definem as dimensões da imagem.

    <ul> , <ol> e <li> :

    • Oque faz:
      • <ul>: Cria uma lista não ordenada (com marcadores).</li>
      • <ol>: Cria uma lista ordenada (numerada).
      • <li>: Define um item de lista.
    • Por que é importante: Organiza informações em listas claras e fáceis de ler.
    • Atributos: Podem usar atributos globais como class e id.

    <div> e <span> :

    • O que faz:
      • <div>: Cria um container de bloco para agrupar elementos.</li>
      • <span>: Cria um container em linha para estilizar partes de texto.
    • Por que é importante:Ajuda a organizar e aplicar estilos ao conteúdo.
    • Atributos: Podem usar atributos globais como class e id.

    <mark> :

    • O que faz: Destaca o texto como se estivesse marcado com um marcador.
    • Por que é importante: Para chamar a atenção para partes importantes do texto.
    • Atributos: Podem usar atributos globais como class e id.

    <small> :

    • O que faz: Exibe o texto em um tamanho menor.
    • Por que é importante: Para notas de rodapé ou informações menos importantes.
    • Atributos: Podem usar atributos globais como class e id.

    <i> :

    • O que faz: Exibe o texto em itálico.
    • Por que é importante: Para enfatizar palavras ou frases, ou para termos técnicos.
    • Atributos: Podem usar atributos globais como class e id.

    <u> :

    • O que faz: Sublinha o texto.
    • Por que é importante: Para destacar texto, mas com cuidado, pois pode ser confundido com links.
    • Atributos: Podem usar atributos globais como class e id.

    <strong> :

    • O que faz: Exibe o texto em negrito, indicando importância.
    • Por que é importante: Para destacar partes importantes do texto.
    • Atributos: Podem usar atributos globais como class e id.

    <sub> e <sup> :

    • O que faz:
      • <sub>: Exibe texto como subscrito (abaixo da linha base).
      • <sup>: Exibe texto como sobrescrito (acima da linha base).
    • Por que é importante: Para fórmulas químicas, matemáticas, etc.
    • Atributos: Podem usar atributos globais como class e id.

    <blockquote> :

    • O que faz: Destaca uma citação em bloco.
    • Por que é importante: Para citar textos longos de outras fontes.
    • Atributos: Podem usar atributos globais como class e id.

    <abbr> :

    • O que faz: Define uma abreviação ou acrônimo.
    • Por que é importante: Para fornecer a explicação completa de uma abreviação.
    • Principais Atributos: title: Fornece a explicação completa da abreviação.

    <hr> :

    • O que faz: Insere uma linha horizontal para separar conteúdo.
    • Por que é importante: Para dividir seções de conteúdo.
    • Atributos: Podem usar atributos globais como class e id.
